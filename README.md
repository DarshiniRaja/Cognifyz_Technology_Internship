# Cognifyz Technologies – Software Development Internship 🚀

## 📌 Internship Overview

This repository contains all the tasks I completed during my **Software Development Internship** at **Cognifyz Technology** using Python.  
Each task is designed to enhance hands-on coding skills ranging from basic conditionals to advanced web scraping and file handling.

---

## ✅ Completed Tasks

### 🔹 Level 1: Beginner

#### 🧠 Task 1: Quiz / Guessing Game
- A simple text-based game using conditional statements
- Input-based interactive game with score logic
- **Skill Gained:** Control Flow using `if`, `elif`, `else`

#### 🔢 Task 2: Number Pattern Generator
- Generated 5 different patterns (Pyramid, Diamond, etc.)
- User inputs number → all patterns printed in sequence
- **Skill Gained:** Nested loops & pattern logic

---

### 🔹 Level 2: Intermediate

#### 📋 Task 3: Console-Based CRUD App
- Basic Task Manager using OOP & lists
- Functions for Create, Read, Update, and Delete
- **Skill Gained:** OOP concepts and data handling

#### 🌡️ Task 4: Temperature Converter
- Converts between Celsius & Fahrenheit
- Users choose direction & input values
- **Skill Gained:** Arithmetic ops + user-friendly flow

---

### 🔹 Level 3: Advanced

#### 📁 Task 5: File-Based CRUD App
- Enhanced Task Manager with File I/O
- Stores and loads tasks from a `tasks.txt` file
- **Skill Gained:** File handling (`read`, `write`), data persistence

#### 🕸️ Task 6: Web Scraping Program
- Scrapes quotes from [http://quotes.toscrape.com](http://quotes.toscrape.com)
- Extracts Quote text, Author, and Tags
- **Skill Gained:** Web scraping using `requests` and `BeautifulSoup`

---
